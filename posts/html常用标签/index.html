HTML常用标签

2019年11月8日

《HTML 常用标签》

a 标签

href    超链接

填入相应的链接地址

1.  网址    推荐使用这种格式（//google.com）因为会自动选择协议，跳转到正确页面，减少 bug。

2.  绝对路径于相对路径

注:在哪里开启的 http 服务 哪里就是根目录

3.  伪协议

javascript:代码;    （可实现无任何操作的 a 标签 不加代码）

mailto:  发邮件

tel:  打电话

4.  #id   定位到页面中的 id 位置

target

  • _blank    新建窗口打开
  • _top    级别最高的窗口打开
  • _parent    父级窗口打开
  • _self    默认值    当前窗口打开
  • target="xxx” 创建一个 xxx 窗口 会覆盖之前的窗口
  • iframe name="xxx” 用 name 为 xxx 的 iframe 窗口打开

download

下载页面  部分浏览器不支持

rel=noopener

现在还不清楚具体用处

总结

a 标签的作用：跳转外部页面，内部锚点，邮箱和电话

MDN a 标签

img 标签

原理：发出一个 get 请求 展示一张图片

alt

替代文本

width height

宽高

src

图片路径

onload 事件

onerror 事件

图片加载失败后 展示一张准备好的报错图片

响应式

给 img 设置最大宽度

max-width:100%;

table 标签

牢记：table 的子元素只能是 thead tbody tfoot

tr

table row 表示表的一行

th

table head 表头 （有加粗 居中样式）

td

table data 数据

表格样式

table-layout

默认值：auto   

fixed 平均宽度

border-spacing

边框间隙 一般设置为 0

border-collapse

值为：collapse 表示合并边框
