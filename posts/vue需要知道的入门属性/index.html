Vue需要知道的入门属性

2020年3月20日

这里介绍一下Vue比较简单的，也是一定要知道的属性

el - 挂载点

new Vue({
    el : '#App'
})

//可以使用$mount代替

new Vue({

}).$mount('#App')

data - 内部数据

注意: 在组件中使用data必须是函数

new Vue({
    data(){
        return {
            data : 0
        }
    }
})

methods - 方法

new Vue({
    data{
        data : 0
    }

    methods : {
        add(){
            this.data++
        }
    }
})

this.add()   //1

components - 组件

  • 引入一个vue文件，就是一个组件

  • 声明

Vue.component('Demo',{
    template : `<div>组件</div> `
})
  • 在实例中添加component对象

使用方法

<Demo/>

四个钩子

  • created

当实例出现在内存时执行

  • mounted

当实例出现在页面时执行

  • updated

实例更新后执行

  • destroyed

实例消亡

相应的钩子还有

beforeCreate、beforeMount、beforeUpdate、beforeDestroy

props - 外部数据

//父组件传了一个data

<template>
    <div>
        {data}
    </div>
</template>

export default {
    props : ['data']
}

在传data的时候

<Demo data='n' />  //传了字符串n
<Demo :data='n' />  //传了变量n
没有标签
