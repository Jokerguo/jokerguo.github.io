Sync修饰符

2020年3月5日

理解vue的.sync修饰符

.sync会被扩展为一个自动更新父组件属性的 v-on 监听器，是一个语法糖的存在

代码示例

    <Child :money.sync = "total"/>

    //扩展为

    <Child :money = "total" @update:money = "total = $event"/>

    //子组件要触发的事件 (需要100元)

    <button @click = "$emit('update:money',total - 100)"></button>

子组件向父组件发了一个update:money的方法，父组件监听了这个方法，然后更新了prop

//index.vue

    <template>
        <div>
            App.vue 我现在有 {{total}}
            <hr>
            <Child :money.sync="total"/>
        </div>
    </template>

    <script>
        import Child from "./child.vue";
        export default {
        data() {
            return { total: 10000 };
        },
        components: { Child: Child }
        };
    </script>

//child.vue

    <template>
        <div>
            <button @click="$emit('update:money', money-100)">花钱</button>
        </div>
    </template>

    <script>
        export default {
            props: ["money"]
        };
    </script>
